# Idea Sync Backend
This is a backend application for the Idea Sync project.

## Tech Stack
- springboot
- spring-data-jpa
- spring-web
- mysql

## Getting Started
1. Clone the repository.
2. Create .env for application properties.
    ```bash
    DB_NAME=
    DB_USERNAME=
    DB_PASSWORD=
    ```
3. Load the gradle dependencies, and you are good to go.
